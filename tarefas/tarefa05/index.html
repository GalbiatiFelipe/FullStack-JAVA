Adivinhe o Número!

Descriçao

O jogo é muito simples, assim que você entrou neste link este site pensou em um número, o que você precisa fazer é adivinhar o número que o computador pensou, você terá 5 tentativas para adivinhar e a cada uma delas uma dica irá aparecer dizendo se o número chutado é maior ou menor que o número secreto.
